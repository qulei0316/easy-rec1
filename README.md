# 推荐系统项目（简要说明）

本项目是一个端到端的推荐系统流水线，包含数据生成、特征构建、多阶段召回/排序以及融合等模块。适合用于离线实验、能力验证与流程演示。

## 目录结构
- data/：数据生成脚本
- features/：特征构建
- recall/：召回阶段
- prerank/：粗排阶段
- rank/：精排阶段（含 DNN 版本）
- rerank/：重排阶段
- blend/：多路结果融合
- docs/：需求/设计/架构/部署等文档

## 运行方式
- 安装依赖：参考 requirements.txt
- 一键运行：执行 run_all.sh
