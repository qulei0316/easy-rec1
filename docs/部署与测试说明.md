# 部署与测试说明

## 环境要求
- Python 3.10+ (miniconda/uv)
- pip install -r requirements.txt

## 运行流程
1. 生成数据：python data/generate_data.py
2. 特征工程：python features/build_features.py
3. 召回/粗排/精排/重排/融合：bash run_all.sh（或 python run_all.py）
4. 启动服务：python service/app.py
5. 测试：python tests/test_api.py

## 本地Mac兼容，无需GPU
- macOS 建议使用 tensorflow-macos 版本（已在 requirements.txt 固定）
- 召回向量模型使用 implicit(ALS) + FAISS，纯CPU可运行
- 精排默认运行 Transformer 版本，如需旧版 DNN 请执行 rank/rank_main_dnn.py
